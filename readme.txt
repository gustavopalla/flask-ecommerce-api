API de E-commerce com Flask

Este projeto consiste no desenvolvimento de uma API para simular um sistema de e-commerce, 
utilizando o microframework Flask. A aplicação permite realizar operações essenciais como 
listagem de produtos, adição de itens ao carrinho e finalização de compras (checkout), além 
de incluir autenticação de usuários.

Funcionalidades:

- Cadastro e login de usuários com autenticação JWT;
- CRUD de produtos;
- Adição e remoção de itens no carrinho de compras;
- Finalização de pedidos (checkout);
- Integração com banco de dados relacional;

Tecnologias utilizadas:

- Python;
- Flask;
- Flask-JWT-Extended;
- SQLAlchemy;
- SQLite (pode ser facilmente adaptado para PostgreSQL ou outro banco relacional);
- Postman (para testes de rotas).
